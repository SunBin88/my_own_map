# my_own_map

# Gangwon Conqueror App

Gangwon Conqueror App은 사용자가 강원도의 정복한 지역을 추적하고, 진행 상황을 시각적으로 확인할 수 있는 간단한 안드로이드 애플리케이션입니다.

## 기능

- 강원도를 정복하는 목표를 설정하고 진행 상황을 추적
- 사용자가 정복한 지역의 퍼센티지를 표시
- 버튼을 클릭하여 강원도 정복 상태를 업데이트
