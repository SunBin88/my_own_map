# my_own_map


My Own Map App은 사용자가 강원도의 여행한 지역을 추적하고, 진행 상황을 시각적으로 확인할 수 있는 간단한 안드로이드 애플리케이션입니다.

## 기능

- 강원도를 여행하는 진행 상황을 추적
- 사용자가 여행한 지역의 퍼센티지를 표시
- 버튼을 클릭하여 강원도 여행 상태를 업데이트
